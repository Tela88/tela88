# Agency Memory

## Missão

Construir uma agência de marketing automatizada, com subagentes especializados, memória persistente, skills adequadas e workflows reutilizáveis.

## Preferências iniciais

- automação máxima
- pouca ou nenhuma intervenção manual
- agentes com responsabilidades bem definidas
- outputs consistentes e fáceis de rever

## Próximos objetivos

- definir o sistema de subagentes
- definir a memória por agente
- instalar e organizar MCPs úteis
- criar playbooks de produção
