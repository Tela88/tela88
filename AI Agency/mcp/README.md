# MCP Stack

Lista operacional dos MCPs e conectores que a agência pode vir a usar.

## Categorias

- design
- imagem
- motion
- scraping e browser
- automação
- documentos e reporting
- publishing

## Nota

Instalar tudo de uma vez raramente é a melhor opção.

A abordagem recomendada é:

1. definir os fluxos da agência
2. mapear ferramentas por fluxo
3. instalar primeiro os MCPs que desbloqueiam produção real
