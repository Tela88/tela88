# Clients

Cada cliente deve ter uma pasta própria.

## Estrutura recomendada

- `profile.md` para contexto do negócio
- `brand.md` para identidade, tom e assets
- `offers.md` para serviços, pricing e diferenciação
- `memory.md` para decisões persistentes
- `runs/` para execuções por squad

## Exemplo

`clients/tela-88/`
