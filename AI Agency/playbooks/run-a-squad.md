# Run a Squad

## Passos

1. Escolher o cliente em `clients/`
2. Escolher o squad em `squads/`
3. Criar um `run_id`
4. Copiar `AI Agency/_core/run-template.json`
5. Criar a pasta `clients/<cliente>/runs/<run_id>/`
6. Criar `brief.md`
7. Criar `artifacts/` e `logs/`
8. Atualizar `run.json` a cada mudança de estado

## Estrutura mínima do run

- `run.json`
- `brief.md`
- `artifacts/`
- `logs/`
