# Memory Policy

## Camadas de memória

- `memory/agency-memory.md` para regras e objetivos da agência
- `clients/<cliente>/memory.md` para preferências e decisões persistentes de cada cliente
- `squads/<squad>/memory.md` para aprendizagem útil do pipeline

## O que guardar

- decisões aprovadas
- assets escolhidos
- preferências recorrentes
- erros a evitar

## O que não guardar

- segredos
- tokens
- ruído temporário sem valor futuro
